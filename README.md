# Radiant Rovers Football Academy – Player Performance Analysis

## Project Overview
This project analyzes player-level match performance data from Radiant Rovers Football Academy across 20 competitive matches involving 48 players. 
The analysis applies Python-based data cleaning, PostgreSQL KPI computation, and Power BI visualization to support data-driven coaching and player development decisions.

## Tools & Technologies
- Python (Pandas, NumPy)
- PostgreSQL (SQL)
- Power BI
- Jupyter Notebook

## Dataset Summary
- Players: 48
- Matches: 20
- Granularity: Player–match level

## Workflow
1. Data collected manually during matches by the coach
2. Data cleaning and feature engineering using Python
3. KPI computation using PostgreSQL
4. Interactive dashboard creation using Power BI

## Key KPIs
- Total goals and assists
- Goals per 90 minutes
- Average goals per match (player-sum)
- Passing accuracy by position
- Fitness score trends by age
- Attendance-based consistency

## Deliverables
- Cleaned CSV dataset
- Jupyter Notebook (data preprocessing)
- PostgreSQL KPI queries
- Power BI dashboard
- Final analytical report
